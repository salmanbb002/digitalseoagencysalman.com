# Digital SEO Agency Salman

Static website for `digitalseoagencysalman.com`.

## Files

- `index.html`: Main landing page with SEO metadata and schema.
- `styles.css`: Responsive visual design.
- `script.js`: Mobile navigation, audit estimator, and form note behavior.
- `assets/seo-analytics-workspace.png`: Local hero image generated for the project.
- `competitor-analysis.md`: Research and design notes.

## Preview

Open `index.html` in a browser, or run a simple local server from this folder:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.
